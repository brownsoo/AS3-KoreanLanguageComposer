HangleUnicodeComposer

compare3Syllables()method 
public function compare3Syllables(init:String, mid:String, fin:String):Array

3개 자모를 초성 중성 종성 으로 비교하여 결합할 수 있는 부분은 하나로 결합하여 배열로 리턴시킨다. if three charaters are able to compose with each other, it composes three characters to one. Or it composes two characters to one and left one jamo(jaeum or moeum). And it returns a array that contains one composed string or two string(composed string, fin) or three string(init, mid, fin).

Parameters

init:String — 초성이 되는 글자
 
mid:String — 중성이 되는 글자
 
fin:String — 종성이 되는 글자
